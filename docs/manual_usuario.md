# Manual de usuario

Guía rápida para usar el sistema **Control de Estudiantes**.

## Pantalla principal

Al abrir la aplicación se muestra la **lista de estudiantes** con:

- Tarjetas de estadísticas (total, activos, inactivos, cursos).
- Tabla con los estudiantes registrados.
- Paginación (10 estudiantes por página).

## Registrar un estudiante

1. Hacer clic en **"Nuevo estudiante"**.
2. Completar el formulario (nombre, apellido, email, curso, estado).
3. Guardar. El sistema valida que el email sea único.

## Editar un estudiante

1. En la tabla, hacer clic en **Editar** en la fila del estudiante.
2. Modificar los datos y guardar.

## Eliminar un estudiante

1. Hacer clic en **Eliminar**.
2. Confirmar la eliminación.

## Búsqueda y filtro (solo V2)

- **Búsqueda:** escribir en la barra para buscar por nombre, apellido o email.
- **Filtro por curso:** seleccionar un curso para ver solo sus estudiantes.

## Exportar reportes (solo V2)

Desde la lista se puede exportar a **PDF** o **Excel** con los botones
correspondientes.
